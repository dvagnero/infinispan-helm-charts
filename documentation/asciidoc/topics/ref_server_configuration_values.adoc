= {brandname} Server configuration values

[role="_abstract"]
{brandname} Server configuration values let you do the following:

* Create caches at startup with any {brandname} configuration.
* Add cache templates and use them to create caches on demand.
* Specify JGroups cluster tranport properties.
* Configure Cache Managers with server hinting and capacity factors.
* Define {brandname} Server security realms.
* Configure {brandname} Server endpoints with custom Hot Rod and REST connectors.
* Add {brandname} Server data sources and credential keystores.

[IMPORTANT]
====
To avoid unexpected behavior and loss of service you should not modify some default configuration for {brandname} clusters.

You should not modify or disable:

* Socket bindings named "default" or "metrics".
* Security realm named "metrics".
* Endpoint named "metrics".
====
